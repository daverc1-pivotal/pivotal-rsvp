Join Pivotal for a Cloud Foundry Workshop
on June 10th
 

Join Pivotal technologists and learn how to build and deploy great software on a modern cloud platform.

At this workshop, you’ll get hands-on experience as you test-drive Pivotal’s Cloud Foundry platform. In just a few hours, you’ll realize the speed and ease of pushing an application on Pivotal Cloud Foundry, the world's leading enterprise Cloud Native Platform, powered by Cloud Foundry.
These hands-on sessions, classroom style, will help individuals to get familiar with Pivotal Cloud Foundry basics, architecture and fundamentals of pushing code and building applications.
Seating is limited. First come, first serve.

For more information visit Pivotal Cloud Foundry
